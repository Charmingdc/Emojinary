## Overview
Dive into Emojinary, a captivating web game where you decipher clever emoji puzzles! Built with React and TypeScript, this frontend application offers a delightful and engaging experience, challenging your wit with both classic and daily game modes.

## Features
*   **Engaging Emoji Puzzles**: Test your knowledge and wit by deciphering a wide array of emoji combinations.
*   **Two Exciting Game Modes**:
    *   **Classic Mode**: Play endlessly to sharpen your emoji-solving skills.
    *   **Daily Challenge**: A new puzzle every day for a fresh challenge.
*   **Intuitive User Interface**: A clean and responsive design built with Tailwind CSS ensures a smooth experience across devices.
*   **Sound Control**: Toggle in-game sounds on or off for a personalized gaming environment, with preferences saved locally.
*   **Client-Side Routing**: Seamless navigation between home, game modes, and other sections powered by React Router DOM.
*   **Local Storage Integration**: Persistently stores user preferences like sound settings.

## Getting Started

To get Emojinary up and running on your local machine, follow these steps.

### Installation
Clone the repository:
```bash
git clone git@github.com:Charmingdc/Emojinary
```

Navigate into the project directory:
```bash
cd Emojinary
```

Install the required dependencies:
```bash
npm install
# or
yarn install
```

### Running the Development Server
Start the development server:
```bash
npm run dev
# or
yarn dev
```
ðŸŽ‰ The application will now be running on `http://localhost:5173` (or another port if 5173 is in use).

### Building for Production
To create a production-ready build:
```bash
npm run build
# or
yarn build
```
This will compile the application into the `dist` directory, ready for deployment.

## Usage
Once the development server is running, open your browser and navigate to `http://localhost:5173`.

*   **Welcome Screen**: You'll be greeted with the welcome screen where you can choose a game mode.
*   **Selecting a Mode**:
    *   Click "Classic" to start an endless game.
    *   Click "Daily" to try the puzzle of the day.
*   **Sound Settings**: On the welcome screen, you can toggle the sound on or off using the dedicated button. Your preference will be saved for future sessions.
*   **Navigation**: Use the top bar to easily navigate back to the home screen.
